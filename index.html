Intro Usage FAQ

What is Version Builder?

Version Builder is a Jira plugin that creates a Jira release and automatically links relevant issues based on Git commit messages. It helps streamline your release process.

How to Use

  1. Install the Version Builder plugin from the Atlassian Marketplace.
  2. Connect your GitHub organization and configure the repository filters.
  3. When you create a new release in Jira, Version Builder scans commit messages between selected tags or commits.
  4. It extracts Jira issue keys from commit messages and automatically associates them with the release.

FAQ

What commit format does it support?

Commit messages should include valid Jira ticket keys (e.g., `ABC-123`) anywhere in the message.

Can I filter which repositories to scan?

Yes, you can configure repository inclusion/exclusion in the app settings.
