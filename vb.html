Frequently Asked Questions

How do I connect my GitHub account?

After installing the app, go to the setup page and follow the OAuth prompt to authorize GitHub access.

Can I use different repos for different Jira projects?

Yes! You can configure a different set of repositories per project. Version Builder remembers your last configuration for future convenience.

What issue key format does it detect?

Version Builder uses a standard regex to extract Jira issue keys like ABC-123 from commit messages.

Is the app secure?

Yes. Version Builder uses OAuth to authenticate with GitHub and follows Atlassian Forge's security standards for app development.
