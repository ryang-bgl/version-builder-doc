Common Issues

GitHub Connection Issues

Having trouble connecting your GitHub account? Make sure you have the necessary permissions and try refreshing the page after connection.

Issues Not Being Linked

Ensure your commit messages include proper issue references (e.g., RT-2, RT-3) and that your repository filters are configured correctly.

Missing Repositories

If repositories don't appear, check your GitHub connection and repository filter settings in the Version Builder Repository Setup page.
